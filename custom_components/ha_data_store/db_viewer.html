📖 API 使用说明

基础路径
/api/ha_data_store/query
查询类型 (type)
device_history设备开关记录 · entity_id/date/month/year/room/limit
device_summary设备汇总 · entity_id/date/month/year/room
env_history环境历史 · entity_id/metric/date/room/limit
env_latest环境最新 · entity_id/metric
attr_history属性历史 · attr_type/entity_id/date/room/limit
attr_latest属性最新 · attr_type/entity_id
rooms_daily全部房间（按日）· category/date/attr_type 或 metric · 返回按房间分组的数据
rooms_multi_metric多指标按日汇总 · date/metrics · 返回按房间×指标分组的温度/湿度等数据
vacuum_history扫地机器人轨迹 · vacuum_id/date/limit · 返回坐标+状态序列
entities实体列表 · category/room
entity_data_dates设备有数据的月份查询 · entity_id/month/date_field · 返回指定月份哪些日期有数据
aggregate_daily指定月每日汇总 · entity_id/month/date_field · 返回每日 duration/energy_consumed 汇总
aggregate_monthly指定年每月汇总 · entity_id/year/date_field · 返回每月 duration/energy_consumed 汇总
aggregate_yearly所有年年度汇总 · entity_id/date_field · 返回每年 duration/energy_consumed 汇总
ranking_daily日排行榜 · date/limit · 按 entity_id 汇总 duration/energy，按 duration 降序
ranking_monthly月排行榜 · month/limit · 按 entity_id 汇总 duration/energy，按 duration 降序
ranking_yearly年排行榜 · year/limit · 按 entity_id 汇总 duration/energy，按 duration 降序
electricity_standard标准电费数据 · entity_id（必填）/day_table/month_table/year_table/条数限制 · 返回 state+attributes 结构
health_history健康记录 · name/type/start/end/limit/offset/order_by · 返回健康记录列表
health_latest健康最新 · name（必填） · 返回某人最新一条健康记录
rooms_multi_metric 参数（新增）
date=2025-01-15 — 查询日期（必填）
metrics=temperature,humidity — 指标列表，逗号分隔（必填）
limit=500 — 每个房间每种指标最多条数，默认500
返回格式：{"rooms":{"客厅":{"temperature":[...],"humidity":[...]},"卧室":{...}}}
rooms_daily 参数
category=device — 数据类别（device/environment/attribute）
date=2025-01-01 — 查询日期
attr_type=electricity_daily — 属性类型（category=attribute 时）
metric=temperature — 环境指标（category=environment 时）
limit=1000 — 每个房间最多条数，默认1000
entity_data_dates 参数
entity_id=switch.ac — 实体 ID（必填）
month=2025-01 — 月份 YYYY-MM（必填）
date_field=on_time — 日期字段名（可选，不填则自动检测：设备类用 on_time，环境/属性类用 datetime）
返回格式：{"dates":["2025-01-01","2025-01-02",...],"count":5}
electricity_standard 参数
entity_id=... — 实体 ID（必填）
day_table=attr_ele_day — 日电量表名（可选，默认 attr_ele_day）
month_table=attr_ele_month — 月电量表名（可选，默认 attr_ele_month）
year_table=attr_ele_year — 年电量表名（可选，默认 attr_ele_year）
day_limit=30 — 日数据条数限制（可选，0=不限）
month_limit=12 — 月数据条数限制（可选，0=不限）
year_limit=5 — 年数据条数限制（可选，0=不限）
返回标准电费数据结构，包含 state（translated/raw/last_changed/last_updated）和 attributes（日均消费/剩余天数/预付费/date/daylist/monthlist/yearlist/计费标准/数据源/最后同步日期/unit_of_measurement/icon/friendly_name）
aggregate_daily 参数
entity_id=switch.ac — 实体 ID（必填）
month=2025-01 — 月份 YYYY-MM（必填）
date_field=on_time — 日期字段名（可选，不填则自动检测）
返回格式：{"daily_summaries":[{"date":"2025-01-01","on_count":5,"total_energy":12.3,"total_duration":3600},...]}
aggregate_monthly 参数
entity_id=switch.ac — 实体 ID（必填）
year=2025 — 年份 YYYY（必填）
date_field=on_time — 日期字段名（可选，不填则自动检测）
返回格式：{"monthly_summaries":[{"month":"2025-01","on_count":50,"total_energy":120.5,"total_duration":50000},...]}
aggregate_yearly 参数
entity_id=switch.ac — 实体 ID（必填）
date_field=on_time — 日期字段名（可选，不填则自动检测）
返回格式：{"yearly_summaries":[{"year":"2025","on_count":500,"total_energy":1200.5,"total_duration":500000},...]}
通用参数（attr_history / env_history 支持）
fields=day,dayEleNum — 只返回指定字段
order_by=day — 排序，-day 降序
offset=100 — 分页偏移
其他 API
GET /api/ha_data_store/stats — 各表行数+磁盘占用+最后写入
GET /api/ha_data_store/entity_state?entity_id=... — 实体状态+属性树
GET /api/ha_data_store/attr_types — 属性类型列表
GET /api/ha_data_store/db_viewer/data?table=...&page=1 — 分页查表
GET /api/ha_data_store/config — 实体配置
